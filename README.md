# Bestarion AI Operation

React + TypeScript interactive e-learning page built with Vite.

## Deploy with GitHub Pages

1. Upload all files and folders in this package to the GitHub repository `bestarion-ai-operation`.
2. Go to Settings > Pages.
3. In Build and deployment, select Source: GitHub Actions.
4. Open the Actions tab and wait until the workflow finishes successfully.
5. Open the GitHub Pages URL: `https://diuthihieu.github.io/bestarion-ai-operation/`.

## Local preview

```bash
npm install
npm run dev
```
